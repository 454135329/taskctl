# taskctl
Simple time manager

# Usage
Starting new task
```bash
$ taskctl start TSK-42
```

Stoping existing task
```bash
$ taskctl stop TSK-42
```

Listing logged time
```bash
$ taskctl list
+---------+--------+-------------+
|  TASK   | STATUS | LOGGED TIME |
+---------+--------+-------------+
| TSK-42  | STOP   | 1h 20m      |
+---------+--------+-------------+
```
